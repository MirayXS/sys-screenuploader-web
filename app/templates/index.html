sys-screenuploader

This is a forwarder service for sys-screenuploader.

sys-screenuploader is a sysmodule which monitors /Nintendo/Album directory on your SD card for new files and uploads them to this website for further processing.

Implemented forwarding modules

  • Telegram

Feel free to request new modules and send other feature requests/proposals/bugreports (contacts below)

Installation

Download latest sys-screenuploader.zip and extract it to SD card root.

Signup on this website, create a destination and put its ID (visible on the destination list page) to /config/sys-screenuploader/config.ini

If you use Kosmos CFW, you may move 42000062616B6101 directory from /atmosphere/titles to /atmosphere/contents and append following section to /switch/KosmosToolbox/config.json to add this sysmodule to Kosmos Toolbox "Background services" menu:

{"name": "sys-screenuploader", "tid": "42000062616B6101", "requires_reboot": false}
